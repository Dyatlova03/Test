Откройте программу Intellij IDEA для компьютера. Загрузите проект Cinema, проверьте подключение к базе данных.
Если база данных уже подключена, то можно смело нажимать пуск.
Если база данных ещё не подключена, тогда вам нужно будет создать подключение. Это можно сделать, нажав на вкладку
Database с правой стороны окна программы.
Нажать на плюсик "New".
Выбрать пункт Data Source.
Выбрать установленную SQL программу.
Ввести хост, логин, пароль и имя базы данных.
Затем нажмите Test Connection.
Как только увидите положительное завершение команды, можно приступать к запуску программы.

После запуска программы, в консоли вы увидете меню.
Для того чтобы сделать выбор, введите с клавиатуры цифру, соответствующую нужному вам пункту.
Далее следуйте всем указаниям программы.
После завершения одного пункта меню вам предложат выбрать другой или выйти из программы.
